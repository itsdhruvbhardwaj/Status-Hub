\# StatusHub – Status Saver \& Media Manager



StatusHub is a modern Android app built with Jetpack Compose that allows users to view, manage, and save status media (images and videos) from messaging apps like WhatsApp.



\## ✨ Features



\- 📂 Detects viewed status media from local storage

\- 🖼️ Grid-based gallery for images and videos

\- 🔄 Swipe navigation for fullscreen media preview

\- 🎥 Video playback using ExoPlayer

\- 🧭 Smooth tab navigation (Images / Videos / Favorites)

\- 🎨 Modern UI with glassmorphism-inspired design

\- ⚡ Fast performance with optimized loading

\- 🔒 Privacy-first: no data collection or network usage



\## 🚀 Tech Stack



\- Kotlin

\- Jetpack Compose (UI)

\- Media3 ExoPlayer (Video playback)

\- Coil (Image \& video thumbnails)

\- SAF (Storage Access Framework)



\## 📱 Functionality



\- Browse statuses in grid view

\- Tap to open fullscreen preview

\- Swipe left/right to navigate media

\- Back button to exit preview

\- Video and image handling separated for performance



\## 🔐 Permissions



\- Storage access (user-selected via SAF)

\- No background access or data tracking



\## ⚠️ Disclaimer



This app is not affiliated with or endorsed by WhatsApp or any messaging platform. It only accesses media files already stored on the user's device.
